## üî¥ Maven Integration

If you're using Maven, add the following to your pom.xml:

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

```

```xml
<dependencies>
    <dependency>
        <groupId>com.github.fzzgg</groupId>
        <artifactId>Invify-lib</artifactId>
        <version>{VERSION}</version>
    </dependency>
</dependencies>

```
